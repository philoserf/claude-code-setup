# Commit Message Examples

## Good Messages

```text
Add user authentication system

Implement JWT-based auth with bcrypt password hashing.
Includes middleware for protected routes.
```

```text
Fix memory leak in file processor

Close file handles properly in error cases.
Add resource cleanup in finally block.
```

## Poor Messages (Avoid)

```text
WIP
```

```text
fix
```

```text
updates
```

```text
changes
```

## Quality Criteria

- Describes WHAT changed
- Explains WHY (if not obvious)
- Uses imperative mood ("Add" not "Added")
- Has proper capitalization
- Includes Co-Authored-By attribution
